# Sathish R — Portfolio

Full-Stack Developer & AI Specialist portfolio website. Built with vanilla HTML, CSS, and JavaScript.

## Features

- Animated hero section with 3D particle canvas
- Custom cursor and loading animations
- Responsive design (mobile-first)
- Project showcase with filterable categories
- Testimonials and trust indicators
- Contact section with direct links

## Tech Stack

- HTML5
- CSS3 (custom properties, animations, grid/flexbox)
- Vanilla JavaScript (ES6+)
- Canvas API for particle effects

## Getting Started

Clone the repo and open `index.html` in your browser — no build tools needed.

```bash
git clone https://github.com/Sathish22082007/Sathish-Portfolio.git
cd Sathish-Portfolio
```

Open `index.html` in your browser.
